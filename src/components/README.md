# Components

Conventions:

- Each component lives in its own folder with `index.tsx` and optional `styles.css`.
- Use PascalCase for component folders and filenames (`Button`, `Header`).
- Prefer small, focused components and colocated tests in `__tests__`.

Example:

- `src/components/Button/index.tsx`
- `src/components/Button/styles.css`
- `src/components/Button/__tests__/Button.test.tsx`
